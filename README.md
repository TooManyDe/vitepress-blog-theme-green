## Installation & Build

``` 
$ git clone https://github.com/TooManyDe/vitepress-blog-theme-green.git
```
```
cd vitepress-blog-theme-green
```
```
pnpm install
```
```
pnpm docs:build
```
```
pnpm docs:dev
```

## Deploy to Cloudflare Pages

Framework preset: **none**

Build command: **pnpm docs:build**

Build output directory: **docs/.vitepress/dist**
